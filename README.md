
# SkyScan | Umbrella Corp. Terminal

Интерактивный погодный терминал, разработанный по стандартам корпорации Umbrella для оперативного мониторинга атмосферных условий и биологических угроз.

## Ключевые Характеристики

### 1. ИИ-Оракул (DeepSeek Integration)
- **Персонализированные инсайты**: Продвинутая генерация ироничных и интеллектуальных ответов.
- **Безопасность данных**: Полный запрет на упоминание геолокации в ответах для соблюдения режима секретности.
- **Интерактивный чат**: Поддержка уточняющих вопросов через призму интересов корпорации.
- **BIO_MARKER**: Уникальный ID пользователя в виде смертельного заболевания (RU/EN/BY).

### 2. Метеорологический Мониторинг
- **Актуальные данные**: Температура, влажность, давление, Kp-индекс и риск аллергенов.
- **Sat-Orbital Интерфейс**: Интерактивные метео-карты (радар, ветер, температура) через Windy.
- **Log Recovery**: Доступ к архивам погоды за последние 10 суток.
- **Интеллектуальный прогноз**: Почасовая сводка теперь всегда начинается с текущего времени.

### 3. Интерфейс и Звук
- **Ambient Engine**: Динамический фон, адаптирующийся под погоду (включая спец. протокол для дождя).
- **Корпоративный стиль**: Использование официального логотипа Umbrella Corp (`Logo.png`) с эффектом свечения.
- **Audio Engine**: Механические звуковые эффекты реле и атмосферные фоновые звуки.
- **Security Lock**: Экран авторизации Umbrella (пароль: `20_02`).
- **Screensaver Mode**: Полноэкранные часы с погодной сводкой (активация: двойной клик на логотип).

## Развертывание на GitHub

Чтобы обновить проект в вашем репозитории, выполните:

```bash
# 1. Подготовка изменений
git add .

# 2. Фиксация
git commit -m "актуализация интерфейса и логики прогноза"

# 3. Отправка
git push origin main
```

## Локальный запуск

1. **Установите зависимости**: `npm install`
2. **Настройте .env.local**: Добавьте `GOOGLE_GENAI_API_KEY`
3. **Запуск**: `npm run dev`
